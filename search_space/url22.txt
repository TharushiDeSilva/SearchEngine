Java thread analysers, maintained by Viktoriia Kuznetcova. Useful for identifying and investigating performance problems, thread locks and crashes in an application.

(Year in brackets is when the tool/page was last updated)

    (2014) ThreadLogic: Thread Dump Analyzer with added logic for common patterns
    (2014) JProfiler - a commercial Java profiler, includes functionality for live threads monitoring (locally and remotely) as well as for taking and analysing thread dumps
    (2014) VisualVM - free visual tool, can monitor live threads and take thread dumps, locally and remotely
    (2014) YourKit - a commercial tool, can monitor thread activity live and through thread dumps, locally and remotely
    (2014) ManageEngine Applications Manager - a commercial tool, does thread dumps analysis, reports on thread usage
    (2014) MJProf command line tool processes jstack thread dumps
    (2013) Thread Dump Viewer - a tool for thread dump analysis. Opens thread dumps from Oracle HP and IBM JVMs
    (2011) TDA - Thread Dump Analyzer
    (2008) Samurai - GUI tail tool, picks up thread dumps from a file and visually presents it, color coding thread states
